# 👋 안녕하세요! 게임 클라이언트 개발자 임주형입니다.
Unity와 C#을 기반으로 독창적인 게임을 만드는 것을 즐깁니다.  
현재는 OpenAI API를 이용하여 페르소나를 부여한 NPC와의 상호작용을 중점으로 한 게임 개발에 힘쓰고 있습니다.

## 🔧 사용 기술
- Unity, C#
- OpenGL
- OpenCV
- Photon Pun2
- OpenAI API
